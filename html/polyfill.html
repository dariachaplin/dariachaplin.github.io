Polygon Fill Algorithm - IN PROGRESS

This tool demonstrates the scan line polygon fill algorithm. For more in depth information on how the algorithm is implemented, see the Resources tab.

To walk through an example step-by-step, select one of the available shapes with the buttons below the canvas on the left.

Navigate to the Data Structures tab to view the current state of the Edge Table and the Active List.

In order to control the flow of events, utilize the set of buttons underneath the canvas. These allow you to step through the code, updating data structures and filling in each scan line in the shape. Note that the second row of buttons all pertain to making updates to the Active List.

The 'Complete Shape' button will move through the algorithm automatically, pausing briefly between each step to show the updates.

Hints: The flow of buttons is 'Remove Edges', 'Add Edges', 'Reorder', 'Increment Scan Line', and then 'Update X'. Each button must be pressed every round, even if no updates will be made.
